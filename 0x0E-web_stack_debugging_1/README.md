# Web Stack Debugging 1

This project contains a script to configure an Ubuntu container's Nginx installation to listen on port 80.

## Files

- `0-nginx_likes_port_80`: A Bash script to fix and ensure Nginx is running and listening on port 80.

## Requirements

- Ubuntu 20.04 LTS
- Bash scripts must pass Shellcheck
- Scripts must be executable

## Usage

```bash
chmod +x 0-nginx_likes_port_80
./0-nginx_likes_port_80
